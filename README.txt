# Projeto 3 Semestre

Este projeto é uma aplicação web que permite aos funcionários gerenciar os quartos e clientes forma eficiente. 

## Instalação

Para instalar o projeto, siga os seguintes passos:

1. Clone o repositório:

2. Entre no diretório do projeto:
git clone https://github.com/LipeOliver-prog/SistemaTransylvania

3. Instale o software: VisualCode e XAMPP, clone o projeto na pasta htdocs dentro do XAMP, e 
                                Crie uma tabela com o nome de transylvaniaoz no msqladmin usando o XAMP
                                Importe o banco de dados na tablea transylvaniaoz.

4. Inicie o servidor: XAMP (APACHE E MYSQL)


## Exemplos de Uso

Após iniciar o servidor, acesse a URL `http://localhost/login.php` em seu navegador. Você poderá adicionar, editar e excluir tarefas.
